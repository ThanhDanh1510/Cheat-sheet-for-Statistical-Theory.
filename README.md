# Cheat sheet for Statistical Theory

This repository contains LaTeX sources and compiled PDFs for statistical theory cheat sheets.

## Files

- [statistics-equations.tex](statistics-equations.tex): LaTeX source file containing equations and theories.
- [statistics-equations.pdf](statistics-equations.pdf): Compiled PDF file of the cheat sheet.

## Usage

You can use the LaTeX file to modify or add formulas, or download the PDF directly for reference.
